🔐 Password Strength Tester (Python + CustomTkinter)

An interactive desktop app built with Python, Tkinter, and CustomTkinter to help users create stronger, safer passwords. The app evaluates password strength in real time, provides suggestions for improvement, and includes several unique features to make security fun and engaging.

✨ Features
- Strength Meter & Suggestions – Instant feedback with color coding, emojis, and motivational messages.  
- Entropy Calculator – Shows how unpredictable your password is in bits.  
- Breach Checker – Warns if your password matches known leaked ones.  
- Password Generator – Creates secure random passwords with one click.  
- Copy to Clipboard – Quickly copy generated or tested passwords.  
- Dark/Light Mode Toggle – Full UI theme switch with color changes.  
- Tips & About Section – Helpful best practices and app info via menu bar.  
- Interactive UI – Progress bar, expandable suggestions box, and modern design.

 🛠 Tech Stack
- Python 3
- Tkinter & CustomTkinter => for GUI
- Regex & Math => for password logic and entropy calculation

🚀 How It Works
1. Enter a password in the input box.  
2. The app evaluates length, uppercase/lowercase letters, numbers, and special characters.  
3. Suggestions appear in a scrollable box if improvements are needed.  
4. Entropy and breach check results are displayed instantly.  
5. Use the generator, copy, and theme toggle buttons for extra functionality.
